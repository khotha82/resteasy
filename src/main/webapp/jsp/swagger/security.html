Authorization

The HTTP Authorization header with oAuth info as specified in oAuth 1.0 is used for all URIs. Please note this is implemented as 2-legged. Read details here: http://hueniverse.com/oauth/guide/

Authorization: OAuth realm="mhhe",
              oauth_consumer_key="bbim_oauth_key",
              oauth_token="",
              oauth_signature_method="HMAC-SHA1",
              oauth_signature="wOJIO9A2W5mFwDgiDvZbTSMK%2FPY%3D",
              oauth_timestamp="137131200",
              oauth_nonce="4572616e48616d6d65724c61686176",
              oauth_version="1.0"

OAUTH Error Codes

Error Code Error Description
OAPI-OAUTH-000 Invalid header
OAPI-OAUTH-001 Consumer is not valid
OAPI-OAUTH-002 Signature does not match
OAPI-OAUTH-003 Timestamp is different more than +-10 minutes (Configurable for each consumer)
OAPI-OAUTH-004 Nonce is repeated
